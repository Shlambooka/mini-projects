CSS Selectors

Compound Selectors

p.myclass{} Compound: A sequence of simple selectors that are not separated by a combinator.

Combinator Selectors

p .myclass{} Descendant: The p tag is the ancestor of .myclass.

p > .myclass{} Parent-child: The p tag is the parent of .myclass.

p + .myclass{} Adjacent sibling: The p tag is the first sibling of .myclass.

p ~ .myclass{} General sibling: The p tag is any sibling of .myclass.

Attribute Selectors

[title]{} Simple attribute: Selects the title attribute.

[title=value]{} Represents elements with an attribute name of title whose value is exactly value.

[title~=value]{} Represents elements with an attribute name of title whose value is a whitespace-separated list of words, one of which is exactly value.

Pseudo-classes for document structures

Only Selectors

p:only-child{} Selects if the element is an only child (no siblings).

p:only-of-type{} Selects if it is the only element of its type.

First and Last Selectors

p:first-child{} Selects <p> if it is the first child.

p:first-of-type{} Selects <p> if it is the first of type.

p:last-child{} Selects <p> if it is the last child.

p:last-of-type{} Selects <p> if it is the last of type.

nth Selectors

li:nth-child(){} Selects <li> if it is the nth child.

  • li:nth-child(even){} Selects all even elements in a list.

  • li:nth-child(odd){} Selects all odd elements in a list.

  • li:nth-child(6){} Selects the 6th element in a list.

li:nth-child(){} or li:nth-of-type(){} count from the top.

  • li:nth-child(3n + 1){} Selects every third element on the list (3n) and (+) start with the first item (1).

li:nth-last-child(){} or li:nth-last-of-type(){} count from the bottom.

  • li:nth-last-child(3n + 6){} Selects every third element on the list (3n) and (+) start with the 6th last item in the list and count up from there.(6).

Unusual Selectors

:root{} Selects the root of the document (the HTML tag itself).

  • Has a higher level of specificity in your CSS.
  • Accesses the shadow DOM if you use JavaScript.
  • Used with CSS variables/custom properties.

:empty{} Selects empty tags (comments permitted, white spaces are not).

Pseudo-classes for Links

a:link{} Selects unvisited links.

a:visited{} Selects visited links.

a:hover{} Selects link when mouse hovers over them.

a:active{} Selects link when clicking down with mouse.

a:any-link{} Selects any link in the visited or unvisited state - helps with raising specificity in the cascade (not super useful).

a:target{} Selects links within a page and can highlight the linked section (works with href="#" tags).

Accessibility

a:focus{} Selects the tab focus so you can apply styles.

ul:focus-within{} Selects the element when there is a focus in the area.

Pseudo-classes for Forms

Form controls

:required{} Selects elements with the required attribute.

:optional{} Selects elements with the optional attribute.

:enabled{} Selects elements with the enabled attribute.

:disabled{} Selects elements with the disabled attribute.

:read-only{} Selects elements with the read-only attribute.

:read-write{} Selects elements with the read-write attribute.

Form control states

:indeterminate{} Selects button groups that do not have default or checked states.

:default{} Selects checkboxes with the default attribute (and the first submit button in the form).

:checked{} Selects checked elements.

Text fields

:autofill{} Selects elements with the autofill attribute.

:placeholder-shown{} Selects elements with the place-holder attribute.

Error Checking

:valid{} Selects valid entries in the form.

:invalid{} Selects invalid entries in the form.

:in-range{} Selects form fields that are within allowable range in the form (numbers).

:out-of-range{} Selects form fields that are out of range in the form (numbers).
