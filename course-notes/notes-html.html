HTML & CSS Linking

Linking Basics

Here is a link going home.

To create a link, add the anchor tag <a> with the href="" attribute.

<a href="../index.html">Link</a>

Local Directory Linking

../ means you are navigating to the parent directory.

./ means you are navigating to the current directory itself.

/ means you are navigating to the root directory.

Offsite Linking

target="_blank" means you are navigating to a new tab (offsite)

<iframe src="/index.html"> this is an iframe.

It is a security risk when you try to open a link pointing offsite.

It allows you to embed content from elsewhere.

target="_parent" means the link will open inside of the iframes' parent, which is the webpage you are on.
