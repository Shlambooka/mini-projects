HTML & CSS Linking

Linking Basics

Here is a link going home.

To create a link, add the anchor tag <a> with the href="" attribute.

<a href="../index.html">Link</a>

Local Directory Linking

../ means you are navigating to the parent directory.

./ means you are navigating to the current directory itself.

/ means you are navigating to the root directory.

Offsite Linking

target="_blank" means you are navigating to a new tab (offsite)

<iframe src="/index.html"> this is an iframe.

It is a security risk when you try to open a link pointing offsite.

It allows you to embed content from elsewhere.

target="_parent" means the link will open inside of the iframes' parent, which is the webpage you are on.

Linking PDFs

<a href="/pdfs/dummy.pdf.html">Dummy(click to download)

You can also link a pdf via a url to an offsite document.

Linking Within a Document

<a href="#top">Back to top will take you to the <div id="top"> element that sits at the top of this webpage.

Linking Phone Numbers

<a href="tel:1-234-567-8910">Call me at 1-234-567-8910

Linking Email

<a href="mailto:email@email.com?subject=addRandomSubject&cc=another@email.com&body=I can even write in the body">Email me

  • add ?subject= to add a subject to the email.
  • add &cc= to add another email address to the email.
  • add &body= to add text to the body in the email.
